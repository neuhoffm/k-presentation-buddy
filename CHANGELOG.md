# Changelog

All notable changes to K Presentation Buddy will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-06

### Added
- Initial release of K Presentation Buddy
- Core presentation automation features
- Terminal integration for demonstrations
- Support for IntelliJ IDEA 2024.1+
- File creation and manipulation capabilities
- Text typing simulation with configurable delays
- Selection and editing operations
- Command execution support

### Changed
- Migrated to MIT License
- Updated to use proper logging instead of println
- Improved code documentation

### Fixed
- Initial bug fixes and stability improvements
