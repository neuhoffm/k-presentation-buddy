# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in K Presentation Buddy, please report it responsibly by emailing
admin@advntrs.com instead of using the public issue tracker.

Include the following information:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any known mitigations

We will acknowledge receipt within 48 hours and provide a status update within one week.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | :white_check_mark: |

## Security Best Practices

When using this plugin:

- Keep your IntelliJ IDEA installation updated
- Update the plugin regularly for security patches
- Report any suspicious behavior immediately

## Dependencies

We monitor dependencies for security vulnerabilities using standard tools and will issue updates promptly when needed.

### Current Direct Build Dependencies

- Kotlin JVM plugin: 2.1.0
- Kotlin serialization plugin: 2.1.0
- kotlinx-serialization-json: 1.6.3
- IntelliJ Platform Gradle Plugin: 2.12.0
- IntelliJ IDEA Community target: 2025.1.1

We regularly review and update dependencies to address known vulnerabilities.
